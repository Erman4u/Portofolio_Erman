# 🎮 Pixel Cyber Portfolio — Erman Saputra

Website portofolio jasa pembuatan website dengan tema **Cyberpunk Pixel** yang modern dan interaktif. Terhubung dengan Google Apps Script sebagai backend untuk mengelola data proyek, pesan kontak, testimoni klien, dan visitor counter secara real-time.

---

## ✨ Fitur Utama

| Fitur | Deskripsi |
|---|---|
| 🌌 Particle Starfield | Animasi partikel bintang di background |
| 📊 Scroll Progress Bar | Garis neon di atas layar mengikuti scroll |
| 🎬 Scroll Reveal | Setiap section muncul dengan animasi saat di-scroll |
| 🃏 3D Tilt Cards | Kartu proyek miring mengikuti kursor |
| ⌨️ Typing Effect | Deskripsi hero diketik otomatis |
| 💻 Terminal Interaktif | Ketik perintah (`help`, `about`, `skills`, dll.) |
| 📬 Contact Form | Form kontak langsung tersimpan ke Google Sheet |
| ⭐ Testimonial | Review klien dikelola dari dashboard GAS |
| 🌗 Dark/Light Mode | Toggle tema gelap dan terang |
| 📄 Download CV | Tombol unduh CV di hero section |
| 👁️ Visitor Counter | Menghitung jumlah pengunjung website |
| 🌊 Wave Footer | SVG wave separator dan ikon sosial media |

---

## 🛠 Tech Stack

- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Backend**: Google Apps Script + Google Sheets
- **Font**: Press Start 2P, Outfit (Google Fonts)
- **Hosting**: GitHub Pages

---

## 📁 Struktur File

```
Web Portofolio/
├── index.html          # Halaman utama
├── style.css           # Styling cyberpunk pixel
├── script.js           # Semua fitur interaktif
├── img/
│   └── Foto.png        # Foto profil
├── Hasil Proyek/       # Screenshot hasil proyek
└── README.md
```

> ⚠️ File backend (`Code.gs` & `Index.html` untuk Google Apps Script) **tidak** disertakan di repository ini demi keamanan.

---

## 🚀 Setup & Deploy

1. Clone repository ini.
2. Buka `script.js` dan ubah `API_URL` dengan URL Web App Google Apps Script Anda.
3. Hosting dengan **GitHub Pages** atau web server pilihan Anda.
